Projects

This page contains an overview of the various projects, I have worked on, arranged by some broad topics. Some of the codes are available here.

Machine Learning

Causal Bandits on General Graphs

Gaurav Sinha (Adobe Research) and Vineet Nair (Technion)

  • Designed an algorithm for simple regret minimization in a general causal graph with unobserved confounders
  • Provided a lower bound on the simple regret for some class of graphs and distributions
  • Designed an algorithm for cumulative regret minimization in a general graph where all nodes are observable
  • A paper based on this work is under review and the preprint is available at arXiv

Intervention Efficient Algorithm for Two-Stage Causal MDPs

Rahul Madhavan (IISc.), Gaurav Sinha (Adobe Research) and Prof. Siddharth Barman (IISc.)

  • Formulated and studied two-stage Causal MDP
  • Developed an algorithm for simple regret minimization, by reducing the problem of exploration and taking advantage of causal information to a convex optimization problem
  • Provided a lower bound for certain transition kernels for the Causal MDP
  • A paper based on this work is under review and the preprint is available at arXiv

Targetability aware Behavioral Segmentation: Incorporating Unobserved Behaviors

Atanu Sinha (Adobe Research), Harshita Chopra, Atishay Ganesh, Sarthak Kapoor, Saili Myana (Research Interns, Adobe Research)

  • Introduced the incorporation of \emphbold{Unobserved Behaviors} and showed its usefulness in behavioral segmentation
  • Showed the \emphbold{predicitive power} of our segmentation framework, so that it can be evaluated against actual outcomes
  • Allowing for \emphbold{mixed targeting} to offer a flexible segmentation approach with UBs

Identifying Roles and Designing Incentives for Unequal Teams in Multi-Agent Systems

Prof. Niloy Ganguly and Prof. Sourangshu Bhattacharya Aug 2019 – Present

  • Designed a game environment based on well-known multi-agent-particle-environment, where two teams (weak, strong) with unequal but changing skill levels compete against each other.
  • Showed the adverse effects of ensuring fairness among differently capable teams using incentive schemes, in a mixed co-operative and competitive environment and proposed ways to mitigate them
  • Proposed a new framework for learning emergent roles which are dynamic and diverse, and uses the actor-critic framework with role encoder and opponent role predictors, that are used in learning an optimal policy

Identification of keyphrases, classification of identified keyphrases and extraction of relationship between two keyphrases

Prof. Plaban Kumar Bhowmick Jan 2019 – Apr 2019

  • Worked on the SemEval 2017 Task 10: Extracting Keyphrases and Relations from Scientific Publications
  • Applied several baseline models including BiLSTM-CRF, CNN and attention-based BiLSTM and also an ensemble of several models to solve the task

Suggestion Mining From Online Reviews and Forums Dec 2018 – Apr 2019

  • Applied several models for Text Classification including Statistical Classifiers, CNN, attention-based BiLSTM, Hierarchical Attention Network, and Graph Convolutional Network (GCN)
  • Tried to overcome the problem of imbalance in the data and out-of-vocabulary words and used semi-supervised approaches, so that a large amount of unlabelled data could be used

Question Generation from RDF graphs via Discriminative Ranking Jul 2018 – Nov 2018

  • Identification of entities and their relations from the knowledge graph generated from RDF Linked Data
  • Extraction of tokens from entities and relationships and generation of a ranked list of natural language questions using Discriminative Ranking

Computer Systems

Auditorium and Room Booking System Jan 2019 – Apr 2019

  • Developed a Web Application using JSP and MySQL to automate room booking in IIT Kharagpur
  • Included dynamic features like submitting/accepting booking request at both the Applicant and Verification Side

Memory-Resident Unix-Like File System Jan 2019 – Apr 2019

  • Created a Multi-Level Directory Tree like File System which supports basic Linux-type file commands
  • iNode Implementation where the Free Blocks are a Linked List and the File Blocks are maintained in iNodes

Reliable User Datagram Protocol Jan 2019 – Apr 2019

  • Created a Static Library with all required functions for our protocol - socket(), send(), recv(), close()
  • Created a Concurrent Thread which managed the Receiving of Messages and placed them into the Receive Buffer
  • This Thread also managed the Acknowledgements and the Re-transmissions to ensure reliability

Tiny-C Compiler Aug 2018 – Oct 2018

  • Implemented a compiler following the language specification based on ISO/IEC 9899:1999 (E) for a subset of C functionalities into x86 Assembly Language
  • Also implemented a Lexical Analyzer using Flex, a Semantic Parser using Bison, and a Machine Independent Code Generator and Translator to convert the Source Code into Three Address Code and finally into x86 Assembly Code

Transport Company Computerization Software Jan 2018 – Apr 2018

  • Developed a software using Java and SQL to automate the processes of a Transport Company
  • Included preparation of SRS, DFD and UML Diagrams, UI development and unit testing of modules
  • Designed and implemented algorithms for inclusion of features like addition, update and removal of branches, employees and vehicles, calculating profit and allotment of vehicles and consignment based on their idle time or waiting time
